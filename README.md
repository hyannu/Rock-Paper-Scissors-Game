 🪨 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.
The game allows users to play against the computer,
keeps track of scores, and displays the result of each round in real time.


 ✨ Features

- Play against the computer
- Random computer move generation
- Real-time winner detection
- Live score updates
- Win, lose, and draw messages
- Simple and responsive user interface


 🛠️ Technologies Used
 
- HTML5
- CSS3
- JavaScript (ES6)

 🎮 Game Rules
 
- Rock beats Scissors.
- Scissors beats Paper.
- Paper beats Rock.
- If both players choose the same option, the game is a draw.
